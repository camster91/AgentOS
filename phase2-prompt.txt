## Step 3: CLI runner

Create core/src/main/kotlin/com/agentOS/core/Main.kt:
- fun main() entry point
- Calls AgentOS.initialize()
- Creates InMemoryStorage for each agent
- Instantiates NotesAgent and CalendarAgent
- Registers both in AgentRegistry
- Prints welcome message
- REPL loop:
  - Prompts user with "> "
  - Reads line from stdin
  - If starts with "notes:" or "note " → routes to NotesAgent.onChat()
  - If starts with "calendar:" or "schedule " or "what's on " → routes to CalendarAgent.onChat()
  - If "agents" → lists registered agents
  - If "help" → shows routing instructions
  - If "quit" or "exit" → exits loop
  - Otherwise → tries NotesAgent first, if response contains "I don't understand" tries CalendarAgent
- Print each agent response
- Use runBlocking for coroutine context

Add to core/build.gradle.kts:
```kotlin
application {
    mainClass.set("com.agentOS.core.MainKt")
}
```
Also add `id("application")` to the plugins block in core/build.gradle.kts.
